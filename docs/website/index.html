What is d-jon?

D-jon is a project manager which implements a real time tracking model, d-jon will record the time you spend in every task, and keep a log of every activity you do to complete them. This is very useful because you will be able to correct the path, change resources and control your project.

Using the Multiproject feature you will track all those activities you did not consider from the beginning and could affect your main project.

The idle and working detection systems will help you to track the time because you will never forget to turn on/off your tracker.

The templates will help you to create the same task structure in a faster and controlled way, if your company uses some kind of process standard you might create a task template (define subtasks, status, etc), everytime you use that template d-jon will create all the subtree in just a few clicks. A FULL TASK TREE WITH ONE CLICK! find out more about templates in: Using templates and Creating templates

Features
  • Multi project:You can manage multiple projects at the same time.
  • Real-Time tracking:Track the time for every task in your project using idle and working detection systems.
  • Manual time entries:You can create your own time logs, useful when you don't have your computer close to you.
  • Gantt View: You can see your tasks in a gantt chart.
  • Templates (EE): You can create templates for your tasks, every time you create a task using that template the subtasks will be automatically created for you. Wiki: Using templates and Creating templates
  • Export (EE): export your projects, tasks and time logs and use them in other applications, reports, etc.
